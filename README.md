Instagram User Analytics

Project Overview

This project analyzes Instagram user behavior using SQL and MySQL Workbench to generate business insights for marketing and investor-related decision-making. The analysis focuses on user engagement, customer retention, hashtag trends, contest performance, and detection of potential bot accounts.

Objectives
Identify loyal and inactive users.
Analyze user engagement patterns.
Discover trending hashtags.
Determine optimal marketing campaign timing.
Detect suspicious or bot-like user activity.
Generate actionable business insights from social media data.
Tools & Technologies
SQL
MySQL Workbench
Microsoft Word
Google Drive
Key Analysis Performed
Identified the oldest and most loyal Instagram users.
Found users who never posted content for re-engagement campaigns.
Determined contest winners based on highest-liked posts.
Analyzed the top-performing hashtags.
Identified the best day for marketing campaign launches.
Calculated average posts per user and engagement metrics.
Detected potential bot accounts that liked every photo on the platform.
Business Impact
Supported customer retention and loyalty programs.
Improved marketing campaign targeting.
Provided insights into user engagement and platform growth.
Helped identify suspicious user behavior for further investigation.
